## 售前咨询文档

### 1. 品种选择与适合性 (15个问题)

**Q1: 我是完全的养猫新手，推荐什么品种？**
**A:** 新手友好三巨头：英短（性格稳定）、美短（体质强健）、中华田园猫（适应力强）。建议从3-6个月已断奶小猫开始，公猫比母猫更亲人。预算2000-5000元足够。

**Q2: 上班族每天早出晚归，养什么猫合适？**
**A:** 推荐独立型品种：英短、俄罗斯蓝猫、异国短毛猫。这些猫咪能较好适应独处。建议准备自动喂食器、饮水机、监控摄像头，每天保证至少1小时互动时间。

**Q3: 家里有老人和孩子，什么猫最温顺？**
**A:** 家庭友好型：布偶猫（忍耐力强）、英国短毛猫（稳重）、异国短毛猫（慵懒温和）。避免选择暹罗猫（敏感）或孟加拉豹猫（过于活跃）。

**Q4: 对猫毛过敏但又想养猫怎么办？**
**A:** 低过敏源选择：1. 斯芬克斯无毛猫 2. 德文卷毛猫 3. 柯尼斯卷毛猫 4. 西伯利亚猫（部分人群）。建议先做过敏测试，家中配空气净化器。

**Q5: 小户型公寓养猫有什么限制？**
**A:** 公寓养猫建议：1. 选择中小型猫（<5kg）2. 利用垂直空间（猫爬架）3. 每天保证20分钟互动 4. 考虑成年猫而非幼猫。缅因等大型猫需要至少20平米活动空间。

**Q6: 预算有限，1000元能买到什么猫？**
**A:** 1000元预算方案：1. 领养中华田园猫（免费-300元）2. 成年英短/美短（800-1200元）3. 轻微瑕疵的品种猫。需预留疫苗300元+用品500元。

**Q7: 想养两只猫作伴，怎么搭配品种？**
**A:** 多猫搭配建议：1. 布偶+布偶（性格相似）2. 英短+美短（都温顺）3. 异性搭配更和谐 4. 年龄差1-2岁最佳。避免两只未绝育公猫。

**Q8: 退休老人适合养什么猫？**
**A:** 老年养猫推荐：1. 成年猫（2岁以上）2. 性格安静的品种 3. 短毛猫（易打理）4. 已绝育疫苗齐全的猫。波斯猫需每天梳理，慎选。

**Q9: 想要不掉毛的猫可能吗？**
**A:** 掉毛程度排序：1. 无毛猫（完全不掉）2. 卷毛猫（掉毛极少）3. 单层被毛猫（掉毛较少）4. 双层被毛猫（掉毛严重）。所有有毛猫都会掉毛，只是多少问题。

**Q10: 公猫母猫哪个更适合新手？**
**A:** 性别对比：公猫更粘人爱叫，体型较大；母猫更独立安静，体型较小。绝育后差异变小。新手建议选择绝育后的公猫。

**Q11: 什么品种的猫最聪明、可训练性强？**
**A:** 聪明度排名：1. 暹罗猫（可训练像狗）2. 阿比西尼亚猫（好奇心强）3. 孟加拉豹猫（学习速度快）4. 布偶猫（可学简单指令）。

**Q12: 喜欢安静的人适合什么猫？**
**A:** 安静型品种：1. 波斯猫（几乎不叫）2. 英国短毛猫（偶尔叫）3. 俄罗斯蓝猫（轻声细语）4. 布偶猫（叫声轻柔）。

**Q13: 猫咪寿命一般多长？什么品种最长寿？**
**A:** 平均寿命12-15年。长寿品种：1. 中华田园猫（15-20年）2. 暹罗猫（15-20年）3. 俄罗斯蓝猫（15-18年）。短寿品种：折耳猫、缅因猫（10-13年）。

**Q14: 学生宿舍可以养猫吗？**
**A:** 宿舍养猫需考虑：1. 宿舍规定 2. 室友同意 3. 寒暑假安置 4. 经济能力。建议养成年中华田园猫，性格稳定花费少。

**Q15: 家里已有狗，养什么猫能和谐相处？**
**A:** 狗友好型猫咪：1. 布偶猫（性格像狗）2. 阿比西尼亚猫（活泼不怕狗）3. 英国短毛猫（淡定从容）。引入需缓慢，全程监督。

### 2. 价格、购买与合同 (15个问题)

**Q16: 为什么布偶猫价格从6000到30000元不等？**
**A:** 价格差异原因：1. 品相血统（赛级vs宠物级）2. 颜色稀有度（海双色vs火焰色）3. 猫舍资质（注册vs后院）4. 地区差异（一线城市更贵）5. 包含服务。

**Q17: 网上买猫如何辨别靠谱卖家？**
**A:** 靠谱卖家特征：1. 可视频看猫实时互动 2. 提供父母照片血统证书 3. 有实体地址可上门 4. 签订正规合同 5. 有健康保证期 6. 售后服务完善。

**Q18: 买猫除了猫钱还要准备多少预算？**
**A:** 初期总预算=猫价+用品(500-1000元)+医疗(1000-2000元)+应急金(2000元)。每月固定开销：猫粮200-500元+猫砂100元+零食玩具100元。

**Q19: 可以分期付款买猫吗？安全吗？**
**A:** 分期付款建议：1. 选择支持分期支付的猫舍 2. 使用第三方平台担保 3. 首付不超过50% 4. 接猫前付清全款 5. 签订分期协议。警惕要求全款预付的卖家。

**Q20: 如何判断猫舍是否正规？**
**A:** 正规猫舍标准：1. CFA/TICA注册 2. 种猫血统可查 3. 小猫绝育后才出售（宠物级）4. 提供疫苗接种记录 5. 有健康保证合同 6. 不过度繁殖。

**Q21: 赛级、宠物级、繁殖级的区别是什么？**
**A:** 等级划分：赛级（符合比赛标准，可比赛繁殖）> 宠物级（微小瑕疵，需绝育）> 繁殖级（可繁殖但不参赛）> 淘汰级（明显缺陷）。新手适合宠物级。

**Q22: 什么季节买猫最划算？**
**A:** 价格季节波动：旺季（春节、暑假，价格上浮10-20%）；淡季（3-4月、9-10月，可能有优惠）。避开猫咪生育低谷期（冬季）。

**Q23: 买猫时如何合理讲价？**
**A:** 讲价技巧：1. 购买多只有优惠 2. 选择成年猫（比幼猫便宜）3. 轻微瑕疵可议价 4. 淡季购买 5. 一次性付全款。避免质疑猫咪健康来压价。

**Q24: 接猫时要做哪些健康检查？**
**A:** 现场检查清单：1. 眼睛清澈无分泌物 2. 鼻子湿润无鼻涕 3. 耳朵干净无异味 4. 口腔牙龈粉红 5. 皮毛光滑无秃斑 6. 肛门干净 7. 精神状态好。

**Q25: 猫咪需要打哪些疫苗？时间表是怎样的？**
**A:** 核心疫苗时间表：8周（第一针猫三联）→ 12周（第二针+狂犬）→ 16周（第三针可选）→ 每年加强。核心防三种病：猫瘟、疱疹、杯状。

**Q26: 健康保证期一般是多久？包含什么？**
**A:** 标准保证：传染病7-30天，遗传病1-2年（需基因检测证明）。包含猫瘟、猫鼻支等重大疾病。不包含：外伤、人为疏忽、已告知的轻微问题。

**Q27: 小猫多大可以接回家最安全？**
**A:** 最佳年龄：12-16周（3-4个月）。前提：已完全断奶、接种第一针疫苗、会用猫砂、吃固体食物、与母猫学会社交技能。

**Q28: 如何准确判断猫咪年龄？**
**A:** 年龄判断法：1. 牙齿：乳牙（2-4周），换牙（4-6个月），牙垢（1-3岁），磨损（5岁以上）2. 眼睛：幼猫清澈，老年可能浑浊 3. 皮毛质地。

**Q29: 绝育有必要吗？最佳时间？c**A:** 绝育必要性：公猫6-8个月，母猫6-8个月。好处：避免意外生育、减少发情行为、降低疾病风险（乳腺肿瘤、子宫蓄脓）。公猫费用300-800元，母猫500-1200元。

**Q30: 宠物保险值得购买吗？怎么选？**
**A:** 保险建议：适合品种易患病（折耳、波斯）、预算充足家庭。选择：1. 覆盖疾病范围广 2. 赔付比例高 3. 等待期短 4. 无年龄限制。年费200-1000元。

### 3. 物流、运输与接猫 (10个问题)

**Q31: 异地购买如何安全运输猫咪？**
**A:** 运输方式：1. 空运（最安全，4小时内，800-1500元）2. 专车（门对门，500公里内）3. 顺风车（经济，需确认接受宠物）4. 铁路（部分城市）。必备：航空箱、饮水器、尿垫、熟悉气味毯子。

**Q32: 接猫前必须准备的物品清单？**
**A:** 必需品：猫砂盆+猫砂、猫粮+食水盆、猫窝/垫子、猫抓板、猫包/航空箱、指甲剪、梳子。可后续添置：猫爬架、玩具、饮水机、自动喂食器。

**Q33: 冬季/夏季接猫特殊注意事项？**
**A:** 冬季：保暖垫、避免室外等待、车内预热、远离暖气。夏季：冰垫、运输时空调、防中暑、充足饮水。共同：减少应激，使用费洛蒙喷雾。

**Q34: 接猫当天详细流程是什么？**
**A:** 流程：确认健康→检查物品→见面核对→支付尾款→签合同→领资料→平稳回家→放入小房间适应→提供食水→给予安静时间。第一晚可能叫唤正常。

**Q35: 接猫后发现健康问题怎么处理？**
**A:** 紧急处理：24小时内联系卖家并拍照留证→7天内健康保证期立即就医→保留所有医疗记录和单据→按合同协商→必要时消协投诉。保留证据是关键。

**Q36: 什么情况下可以退换猫？流程怎样？**
**A:** 可退换：健康期内重大疾病、与合同严重不符、遗传病。不可退换：个人原因、轻微问题（耳螨猫癣）、行为问题。流程：医院证明→联系卖家→按合同处理。

**Q37: 运输中猫咪应激反应严重怎么办？**
**A:** 应激预防：提前熟悉航空箱、铺熟悉气味毯子、费洛蒙喷雾、出发前4小时禁食。运输中：保持平稳安静、适当遮盖、每2-3小时检查。到达后：安静环境适应。

**Q38: 国际运输猫咪需要什么手续？**
**A:** 国际运输流程（提前6个月准备）：1. 狂犬疫苗（30天后）2. 狂犬抗体检测 3. 国际健康证明 4. 芯片植入 5. 航空舱位预订 6. 可能入境隔离。费用1-3万元，建议找代理公司。

**Q39: 猫咪到新家躲着不出来正常吗？多久适应？**
**A:** 完全正常。适应期：1-3天（躲藏）→ 4-7天（试探性出来）→ 2-4周（基本适应）→ 1-3个月（完全安心）。不要强行互动，给予时间和空间。

**Q40: 新猫和原住民如何和平相处？**
**A:** 引入流程：隔离期（1周，分开房间交换气味）→见面期（2-3天，隔着门缝/笼子）→共处期（监督下短暂共处）→完全适应（2-4周）。准备各自资源（食碗、猫砂盆）。

### 4. 喂养、用品与准备 (10个问题)

**Q41: 小猫到家第一天该怎么喂食？**
**A:** 第一天喂养：1. 继续原猫舍同款猫粮 2. 少量多餐 3. 新鲜凉白开 4. 可加益生菌 5. 暂时不给零食 6. 记录进食排便。可能因应激暂时不吃，正常。

**Q42: 猫粮品牌怎么选？进口国产区别大吗？**
**A:** 猫粮选择：看成分（蛋白质>30%，脂肪>15%）。进口高端：渴望、爱肯拿（品质稳定价格贵）。国产优质：诚实一口、网易严选（性价比高）。根据预算选择，能力范围内最好。

**Q43: 猫砂种类太多，哪种最适合新手？**
**A:** 猫砂推荐：新手从豆腐砂开始（可冲厕所、粉尘小、结团好）。进阶选择混合砂（豆腐+膨润土结合优点）。膨润土最经济但粉尘大。松木砂需双层猫砂盆。

**Q44: 智能设备（喂食器、饮水机）有必要吗？**
**A:** 智能设备分析：自动喂食器（适合上班族定时定量，需防卡粮）；饮水机（促进喝水减少尿路问题，需定期清洗）。非必需但提升体验。

**Q45: 猫窝、猫爬架是必需品吗？**
**A:** 猫窝：非必需（很多猫睡床/沙发），可先买基础款试。猫爬架：强烈推荐（提供垂直空间、磨爪、娱乐），尤其多猫家庭和小户型。

**Q46: 必备的猫玩具有哪些？**
**A:** 必备玩具：逗猫棒（互动）、小老鼠（自娱）、猫抓板（保护家具）。可选：激光笔（注意不照眼）、益智玩具、自动玩具。DIY：纸箱、纸团、瓶盖。

**Q47: 家庭药箱需要准备什么？**
**A:** 常备药品：益生菌（肠胃）、化毛膏/猫草（排毛）、体外驱虫药、碘伏棉签。护理：指甲剪、梳子、宠物湿巾、牙刷牙膏。处方药需兽医指导。

**Q48: 猫砂盆选开放式还是封闭式？**
**A:** 猫砂盆选择：开放式（便宜易清理但味大）、封闭式（防臭防溅但有些猫拒绝）、顶入式（几乎不带砂但老年猫不便）。大小原则：猫身长1.5倍。数量：猫数+1。

**Q49: 需要给猫咪买衣服吗？**
**A:** 猫咪衣服：大多数猫不需要（皮毛足够保暖）。特殊情况：无毛猫（需要）、术后保护、冬季外出。选择：透气材质、合适尺寸、无过多装饰。

**Q50: 养猫对家居环境有什么要求？**
**A:** 环境要求：1. 封窗封阳台（防坠落）2. 收起危险品（线绳、药品、植物）3. 预留猫咪活动空间 4. 考虑家具防抓 5. 安静休息区域。提前"猫化"环境。
